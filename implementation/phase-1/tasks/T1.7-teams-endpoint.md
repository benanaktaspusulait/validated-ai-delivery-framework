# T1.7 — GET /api/v1/teams Endpoint

**Phase:** 1  
**Owner:** Platform Engineer  
**Estimate:** 1 day  
**Depends on:** T1.5

## Objective

Expose the `GET /api/v1/teams` endpoint that returns the list of teams with basic info and confidence scores.

## Acceptance criteria

- [ ] Returns 200 with team list.
- [ ] Each team includes: id, name, domain_criticality, data_confidence_score.
- [ ] Supports pagination (limit, offset).
- [ ] Returns total_count, has_more.
- [ ] Returns 401 for unauthenticated requests.

## Technical notes

```text
GET /api/v1/teams?limit=20&offset=0

Response 200:
{
  "teams": [
    {
      "id": "payments-team",
      "name": "Payments Team",
      "domain_criticality": "high",
      "data_confidence_score": 82
    }
  ],
  "total_count": 1,
  "limit": 20,
  "offset": 0,
  "has_more": false
}

Query:
  SELECT t.id, t.name, t.domain_criticality,
         ms.data_confidence_score
  FROM teams t
  LEFT JOIN metric_snapshots ms ON ms.team_id = t.id
    AND ms.metric_name = 'confidence_ai_assisted_pr_rate'
    AND ms.created_at = (SELECT MAX(created_at) FROM metric_snapshots WHERE team_id = t.id AND metric_name = 'confidence_ai_assisted_pr_rate')
  ORDER BY t.name
  LIMIT {limit} OFFSET {offset}
```

## Validation

```text
1. GET /api/v1/teams -> 200, list of teams.
2. Pagination: limit=1, offset=0 -> 1 team, has_more=true.
3. Unauthenticated -> 401.
```
