# T5.10 — Security Pen Test

**Phase:** 5  
**Owner:** Security Lead  
**Estimate:** 1 week  
**Depends on:** T4.1-T4.11

## Objective

Conduct penetration testing before enterprise rollout.

## Acceptance criteria

- [ ] RBAC bypass tests for each role.
- [ ] Webhook signature forgery tests.
- [ ] SQL injection tests on API endpoints.
- [ ] XSS tests on API responses.
- [ ] API rate limiting verification.
- [ ] All critical/high findings fixed.
