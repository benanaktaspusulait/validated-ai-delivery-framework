# T0.7 — HR/People Approval

**Phase:** 0  
**Owner:** HR or People Lead  
**Estimate:** 3 days  
**Depends on:** None

## Objective

Obtain written HR/People approval for the "no individual performance use" policy.

## Acceptance criteria

- [ ] HR/People Lead has reviewed the platform's purpose and data usage.
- [ ] Written acknowledgement of "no individual performance use" policy received.
- [ ] HR/People Lead agrees the platform does not constitute individual surveillance.
- [ ] Approval documented in [Phase0]_exit_report.pdf.

## Technical notes

```text
Key points for HR/People review:
1. The platform reports at team and system level only.
2. Individual rankings, personal dashboards and "who uses AI most" views are prohibited.
3. AI usage metadata is used for team-level learning, never performance scoring.
4. If a manager attempts to use metrics for individual performance, the misuse escalation 
   procedure triggers (docs/governance-and-privacy.md).
5. Developers can challenge metric interpretation.
6. The psychological safety pulse monitors whether the tool creates anxiety.
```
