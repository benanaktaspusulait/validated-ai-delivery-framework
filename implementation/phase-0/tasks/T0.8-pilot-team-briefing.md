# T0.8 — Pilot Team Briefing

**Phase:** 0  
**Owner:** Platform Lead  
**Estimate:** 1 hour  
**Depends on:** T0.6, T0.7

## Objective

Brief the pilot team on the platform's purpose, data usage and non-negotiable cultural rules.

## Acceptance criteria

- [ ] 30-minute briefing session held with all pilot team members.
- [ ] Purpose explained: "We are validating whether the system can use AI safely."
- [ ] Data usage explained: team-level only, no individual scoring.
- [ ] Non-negotiable rules communicated (no ranking, no raw prompts, no hard enforcement before Phase 4).
- [ ] PR template walkthrough completed.
- [ ] Q&A session held.
- [ ] Attendance recorded for [Phase0]_exit_report.pdf.

## Technical notes

```text
Briefing agenda (30 minutes):
1. Purpose (5 min): "We are not measuring individual productivity. We are validating 
   whether the system can use AI safely, sustainably and effectively."
2. What the platform does (5 min): collects PR metadata, computes team-level metrics, 
   shows a dashboard. No personal dashboards. No rankings.
3. What the platform does NOT do (5 min): no individual scoring, no raw prompt storage, 
   no IDE monitoring, no performance reviews.
4. Your role (5 min): fill in the PR template honestly, read in-PR guidance, 
   give feedback in retro.
5. The rules (5 min): non-negotiable principles from docs/governance-and-privacy.md.
6. Q&A (5 min).

Deliverable: attendance list + summary of questions/answers.
```
