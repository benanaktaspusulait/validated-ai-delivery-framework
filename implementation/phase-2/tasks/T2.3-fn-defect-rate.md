# T2.3 — fn_defect_rate_weighted

**Phase:** 2  
**Owner:** Platform Engineer  
**Estimate:** 1 day  
**Depends on:** T1.5

## Objective

Implement the Post-Merge Defect Rate / Weighted Defect Rate metric function.

## Acceptance criteria

- [ ] Counts defects linked to merged PRs in the sprint window.
- [ ] Applies severity weights: cosmetic=0.25, minor=0.5, medium=1.0, major=2.0, critical=5.0.
- [ ] Returns weighted rate = sum(weights) / merged_pr_count.
- [ ] Compares AI vs non-AI cohorts separately.

## Test cases

```text
Fixture: 10 merged PRs, 3 defects (1 minor, 1 medium, 1 major) -> weighted = (0.5+1.0+2.0)/10 = 0.35
Fixture: 0 merged PRs -> rate = 0
Fixture: 10 merged PRs, 0 defects -> rate = 0
```
