# T2.2 — fn_review_debt_ratio

**Phase:** 2  
**Owner:** Platform Engineer  
**Estimate:** 1 day  
**Depends on:** T1.5

## Objective

Implement the AI Review Debt Age Ratio metric function.

## Acceptance criteria

- [ ] Function computes median first-review wait time for non-AI PRs (baseline).
- [ ] Requires minimum 10 non-AI PRs in baseline window. If below: extend window or use platform-wide baseline.
- [ ] Computes average first-review wait time for AI-assisted PRs.
- [ ] Returns ratio = ai_wait / baseline.
- [ ] Returns confidence_issue when baseline is thin.

## Technical notes

```sql
WITH baseline AS (
  SELECT percentile_cont(0.5) WITHIN GROUP (ORDER BY EXTRACT(EPOCH FROM (ra.first_review_at - pr.created_at_source))/3600) AS median_hours
  FROM pull_requests pr
  JOIN review_analytics ra ON ra.pull_request_id = pr.id
  WHERE pr.repository_id IN (SELECT id FROM repositories WHERE team_id = $1)
    AND pr.ai_assisted = false
    AND pr.created_at_source >= $2
    AND ra.first_review_at IS NOT NULL
),
ai_waits AS (
  SELECT AVG(EXTRACT(EPOCH FROM (ra.first_review_at - pr.created_at_source))/3600) AS avg_hours
  FROM pull_requests pr
  JOIN review_analytics ra ON ra.pull_request_id = pr.id
  WHERE pr.repository_id IN (SELECT id FROM repositories WHERE team_id = $1)
    AND (pr.ai_assisted = true OR pr.ai_usage_inferred = true)
    AND pr.created_at_source >= $2
    AND ra.first_review_at IS NOT NULL
)
SELECT
  CASE WHEN b.median_hours = 0 OR b.median_hours IS NULL THEN NULL
       ELSE a.avg_hours / b.median_hours
  END AS metric_value
FROM baseline b, ai_waits a;
```
