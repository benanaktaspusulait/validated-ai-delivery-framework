## Objective

Implement the AI-assisted PR Rate metric function.

## Acceptance criteria

- [ ] Function signature: `fn_ai_pr_rate(team_id, sprint_window) -> { value, confidence_score, confidence_issue }`.
- [ ] Counts AI-assisted PRs (declared_ai + inferred_ai) and total PRs.
- [ ] Excludes unknown PRs from the rate calculation.
- [ ] Returns 0 when no PRs exist (not null).
- [ ] Unit tests pass with fixture data.

## Technical notes

```sql
-- Implementation sketch
WITH team_prs AS (
  SELECT * FROM pull_requests
  WHERE repository_id IN (SELECT id FROM repositories WHERE team_id = $1)
    AND created_at_source >= $2
),
counts AS (
  SELECT
    COUNT(*) FILTER (WHERE ai_assisted = true OR ai_usage_inferred = true) AS ai_count,
    COUNT(*) FILTER (WHERE ai_assisted IS NOT NULL OR ai_usage_inferred IS NOT NULL) AS classified_count
  FROM team_prs
)
SELECT
  CASE WHEN classified_count = 0 THEN 0
       ELSE ai_count::numeric / classified_count
  END AS metric_value
FROM counts;
```
