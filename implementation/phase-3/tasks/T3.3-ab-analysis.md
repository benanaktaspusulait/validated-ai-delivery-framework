# T3.3 — A/B Analysis Engine

**Phase:** 3  
**Owner:** Platform Engineer  
**Estimate:** 2 days  
**Depends on:** T3.2

## Objective

Build the A/B analysis engine that compares treatment vs control outcomes.

## Acceptance criteria

- [ ] Computes: review time, rework rate, defect linkage for each group.
- [ ] Statistical test: Welch's t-test or Mann-Whitney U.
- [ ] Results reproducible (identical inputs -> identical outputs).
- [ ] Report generated for Phase 3 exit report.
- [ ] Confidence interval shown for each metric.

## Technical notes

```text
Input:
  pull_requests WHERE experiment_group IN ('treatment', 'control')
    AND created_at_source >= experiment_start_date

Metrics to compare:
  1. Review time: AVG(first_review_at - created_at_source) per group
  2. Rework rate: AVG(comments_count) per group (proxy for rework)
  3. Defect linkage: COUNT(jira_issues linked to merged PRs) / merged_pr_count per group

Statistical test:
  For each metric:
    treatment_values = [metric for PRs in treatment group]
    control_values = [metric for PRs in control group]

    If n >= 30 per group AND data is approximately normal:
      Use Welch's t-test (unequal variances).
      t_stat, p_value = scipy.stats.ttest_ind(treatment, control, equal_var=False)

    If n < 30 or non-normal:
      Use Mann-Whitney U test (non-parametric).
      u_stat, p_value = scipy.stats.mannwhitneyu(treatment, control, alternative='two-sided')

    Significance level: alpha = 0.05.
    If p_value < 0.05: difference is statistically significant.

Output:
  {
    "experiment_period": "2025-01-01 to 2025-02-15",
    "treatment_n": 245,
    "control_n": 238,
    "metrics": {
      "review_time": {
        "treatment_mean": 14.2,
        "control_mean": 12.8,
        "difference_pct": 10.9,
        "p_value": 0.03,
        "significant": true,
        "interpretation": "Treatment PRs took 10.9% longer to review (statistically significant)"
      },
      "rework_rate": {
        "treatment_mean": 3.2,
        "control_mean": 3.0,
        "difference_pct": 6.7,
        "p_value": 0.42,
        "significant": false,
        "interpretation": "No significant difference in rework rate"
      }
    },
    "recommendation": "Warnings increase review time without quality improvement. Consider softening copy."
  }

Report generation:
  - Export as markdown for inclusion in Phase 3 exit report.
  - Include: sample sizes, means, p-values, interpretation, recommendation.
```

## Test cases

```text
Unit tests:
  - Identical treatment/control -> p_value = 1.0, no significant difference.
  - Treatment significantly worse -> p_value < 0.05, significant = true.
  - Small sample (n=10) -> Mann-Whitney U used instead of t-test.
  - Reproducibility: same inputs -> same outputs (deterministic).

Integration test:
  - Generate 500 fake PRs with known distributions.
  - Run analysis -> results match expected statistical outcomes.
```
