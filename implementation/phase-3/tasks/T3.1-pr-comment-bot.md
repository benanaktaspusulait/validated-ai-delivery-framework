# T3.1 — PR Comment Bot

**Phase:** 3  
**Owner:** Platform Engineer  
**Estimate:** 3 days  
**Depends on:** T2.6

## Objective

Build the PR Comment Bot that posts non-blocking guidance on AI-assisted PRs.

## Acceptance criteria

- [ ] Triggered on pull_request events (opened, edited, synchronize, reopened).
- [ ] Checks for AI metadata completeness.
- [ ] Posts warning for PRs > 300 LOC.
- [ ] Posts risk warning for risk_score >= 10.
- [ ] Posts info for low confidence declarations.
- [ ] Never blocks. Never sets a required check.
- [ ] Comments tracked: pr_bot_comments_total.

## Technical notes

```text
Logic (non-blocking):
  1. If ai_assisted = false (explicitly) -> no comment.
  2. If ai_assisted IS NULL (unknown) -> info comment about missing metadata.
  3. If ai_assisted = true AND changed_lines > 300 -> warning: "Consider splitting".
  4. If ai_assisted = true AND risk_score >= 10 -> warning: "High risk, recommended reviewers".
  5. If ai_assisted = true AND ai_assistance_confidence = "low" -> info: "Ensure you understand the code".

Comment format (GitHub Markdown):
  ## AI Delivery Guidance
  **Status:** {warning|info}
  **Reason:** {specific reason}
  **Recommended actions:**
  - {action 1}
  - {action 2}
  [View in dashboard](link)
```
