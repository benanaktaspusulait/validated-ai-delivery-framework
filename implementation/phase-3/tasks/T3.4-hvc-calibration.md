# T3.4 — Human Validation Cost Calibration Study

**Phase:** 3  
**Owner:** Platform Engineer + Pilot-team reviewers  
**Estimate:** 3 days  
**Depends on:** T2.4

## Objective

Calibrate Human Validation Cost estimates against actual review time.

## Acceptance criteria

- [ ] 20 AI PRs and 20 non-AI PRs selected.
- [ ] Reviewers log actual review time for each PR.
- [ ] Calibration factor computed: actual / estimated.
- [ ] Factor applied to future estimates.
- [ ] Confidence label updated if calibration improves accuracy.

## Technical notes

```text
Study design:
  1. Select 40 PRs from the last 2 sprints:
     - 20 where ai_assisted = true
     - 20 where ai_assisted = false
     - Stratified by size: small (<100 LOC), medium (100-300), large (>300)

  2. For each PR, the reviewer:
     - Starts a timer when they begin reviewing.
     - Stops the timer when they submit their review (approve/request changes).
     - Records actual_hours = stop_time - start_time.

  3. Platform computes estimated_hours for each PR using fn_human_validation_cost.

  4. Calibration factor:
     For each PR: ratio = actual_hours / estimated_hours
     Overall calibration_factor = MEDIAN(ratios)  (use median to reduce outlier influence)

  5. Apply calibration:
     Future estimates: calibrated_hours = estimated_hours * calibration_factor

  6. Confidence improvement:
     If calibration_factor is between 0.7 and 1.3 (estimates are within 30% of actual):
       confidence_label = "calibrated" (upgrade from "directional")
     If calibration_factor is outside 0.7-1.3:
       confidence_label = "directional" (keep current)

Storage:
  ALTER TABLE config_versions ADD COLUMN calibration_factor NUMERIC;
  INSERT INTO config_versions (config_type, config_key, config_value, created_by)
  VALUES ('cost', 'human_validation_calibration_factor', '{"factor": 1.15, "study_date": "2025-02-15"}', 'platform-lead');

  Store individual study results in [Phase3]_exit_report.pdf.
```

## Test cases

```text
Study validation:
  - 40 PRs selected (20 AI, 20 non-AI, stratified by size).
  - All 40 reviews completed with logged times.
  - Calibration factor computed and stored.
  - Future estimates use calibrated factor.
```
