# T3.8 — Slack/Teams Alert Integration

**Phase:** 3  
**Owner:** Platform Engineer  
**Estimate:** 1 day  
**Depends on:** T2.8

## Objective

Route review-debt alerts to a platform Slack/Teams channel.

## Acceptance criteria

- [ ] Alerts sent when AI Review Debt exceeds threshold.
- [ ] Format: "Team {name}: AI Review Debt is at {ratio}x baseline. {N} PRs waiting."
- [ ] Notification, not enforcement.
- [ ] Delivery success tracked.

## Technical notes

```text
Trigger: when AI Review Debt Age Ratio exceeds 1.5 (warning threshold).

Slack integration:
  POST https://hooks.slack.com/services/{webhook_url}
  {
    "channel": "#ai-delivery-alerts",
    "text": "⚠️ *AI Review Debt Alert*\nTeam: {team_name}\nRatio: {ratio}x baseline\nPRs waiting: {count}\nDashboard: {link}"
  }

Teams integration (if applicable):
  POST https://outlook.office.com/webhook/{webhook_url}
  { Adaptive Card format }

Alert frequency: maximum once per team per 4 hours (avoid alert fatigue).
Delivery tracking: notification_delivery_success_rate counter.
```
