# T3.2 — Experiment Mode Assignment

**Phase:** 3  
**Owner:** Platform Engineer  
**Estimate:** 1 day  
**Depends on:** T3.1

## Objective

Implement random assignment of PRs to treatment/control groups for A/B testing.

## Acceptance criteria

- [ ] Each new PR randomly assigned to treatment (50%) or control (50%).
- [ ] Assignment is deterministic (PR number mod 100).
- [ ] Assignment stored in pull_requests table (experiment_group field).
- [ ] Treatment group receives PR comments.
- [ ] Control group receives no PR comments.
