# Enterprise Roadmap

| Stage | Focus | Key additions |
|---|---|---|
| 1 (MVP) | GitHub + Jira, five metrics, team dashboard | Confidence scoring, policy override logging |
| 2 | Platform expansion | GitLab, SonarQube, CI/CD, dynamic WIP, policy-as-code |

| 4 | Intelligence expansion | Trust calibration, maintainability, AI Dependency Risk, Cognitive Load |
| 5 | Enterprise control plane | SSO/RBAC, multi-org, custom policies, API access, regulated-sector packs |
